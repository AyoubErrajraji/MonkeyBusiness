# Monkey Business
A game created with _pygame_ by us!

## Installation
1. Install python 3.6
2. Install pygame:
	`python3.6 pip install pygame`
	Having trouble?
	Try installing pip:
	`wget https://bootstrap.pypa.io/get-pip.py`
	`sudo python3.6 get-pip.py`
	Then redo step 2
3. Clone repository
4. Go to repository directory using `cd`
5. run game:
	`python3.6 main.py`
